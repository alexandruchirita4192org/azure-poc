# AGENTS.md

## Global Instructions

- For Docker usage on an Windows environment, use the free Docker Engine from the WSL2 environment instead of trying to use Docker Desktop which is not free for enterprise environments.
- For Agents, use `C:\Users\[user]\.codex\agents`.
- For Skills, use `C:\Users\[user]\.codex\skills`.
- To determine `[user]`, run `whoami` and use the username portion without the domain prefix.
